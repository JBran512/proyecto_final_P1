## 📝 Descripción del Proyecto
El **Sistema de Administración de Condominio Vista Verde** es una aplicación de escritorio desarrollada para automatizar y optimizar la gestión de complejos residenciales. El software permite centralizar el registro de propietarios y viviendas, controlar los pagos de mantenimiento y generar reportes administrativos detallados. 

Fue desarrollado utilizando **Java Swing** para ofrecer una interfaz gráfica amigable, intuitiva y organizada.

## 🛠️ Instrucciones para Ejecución

### Requisitos Previos:
* Java JDK 17 o superior.
* IDE Apache NetBeans.
* MySQL Server (para la gestión de la base de datos).
* Sistema Operativo Windows 10 o superior.

### Pasos para la instalación y ejecución:
1. **Clonar el repositorio:**
